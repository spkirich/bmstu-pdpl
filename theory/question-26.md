# Математический сопроцессор. Классификация команд

Система команд математического сопроцессора включает:

- Команды пересылки данных:
    - Целочисленные,
    - Действительные,
    - Загрузки констант,
    - Обмена значениями;

- Арифметические команды:
    - Целочисленные,
    - Действительные,
    - Вспомогательные;

- Команды сравнения:
    - Целочисленные,
    - Действительные,
    - Анализ знака,
    - Анализ числа;

- Трансцендентные команды:
    - Тригонометрические,
    - Показательно-логарифмические;

- Команды управления:
    - Инициализирующие,
    - Работы со стеком,
    - Работы со служебными регистрами.
